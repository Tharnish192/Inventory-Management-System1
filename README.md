# Inventory Management System (IMS)

A modern, role-based Inventory Management System built with **Next.js**, **Tailwind CSS**, and **JSON Server**. This application features a premium dark-themed portal and a clean, efficient dashboard for managing stock, sales, and users.

## 🚀 Features

- **Role-Based Access Control (RBAC)**:
  - **Administrator**: Full system control, user management, and detailed analytics.
  - **Staff**: Manage daily operations, process sales, and stock entry.
  - **Viewer**: Read-only access to inventory levels and reports.
- **Inventory Tracking**: Real-time stock levels, low-stock alerts, and product management.
- **Sales & Orders**: Process sale orders and generate purchase orders with ease.
- **Dynamic Analytics**: Visual reports for revenue, stock valuation, and operational history.
- **Responsive Design**: Fast, modern UI matching premium design standards.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend/Database**: [JSON Server](https://github.com/typicode/json-server) (Mock API)

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Tharnish192/Inventory-Management-System1.git
cd Inventory-Management-System1
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the application
Start both the development server and the JSON backend:
```bash
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).
